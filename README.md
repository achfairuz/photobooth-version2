## ✨ Features

- 📷 Real-time camera capture
- 🖼 Elegant templates (Eid Mubarak, Newspaper, etc.)
- 🎨 Photo filters (Vintage, B&W, Romantic, Dramatic)
- 🧩 Multi-frame photobooth system
- 🔁 Retake photo if not aligned
- 💾 High-quality image download
- 🖨 Print-ready output
- 🎉 Confetti animation
- 💎 Modern UI (Glassmorphism + Clean Design)

---

## 🚀 Tech Stack

- ⚛️ React.js
- 🎨 Tailwind CSS
- 📸 HTML Canvas API
- 🧠 html2canvas
- 🎉 canvas-confetti
- 🌐 React Router
